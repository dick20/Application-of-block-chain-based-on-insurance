# Application-of-block-chain-based-on-insurance
基于保险的区块链应用（Solidity智能合约）



使用方法

+ 在根目录下执行npm命令，下载相应的依赖包

  ```
  npm install
  ```

+ 打开testrpc以太坊测试环境

  ```
  testrpc
  ```

+ truffle编译

  ```
  truffle compile
  ```

+ truffle移植

  ```
  truffle migrate
  ```

+ 打开服务器

  ```
  npm run dev
  ```

+ 访问网页

  浏览器输入http://localhost:8080/
